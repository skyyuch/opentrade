## 各基金的關鍵敘事重點

### CCMF（首要目標）

- 屬於「Creative Micro Fund」性質，重視：**創意、可行性、團隊**
- 預期金額小但門檻較低
- **OpenTrade 敘事重點**：
  - Web3 真實落地，不是炒幣項目
  - 解決金融業真實痛點（不公正評分平台）
  - 香港 FinTech + Web3 雙重定位
  - 退休 SFC 董事帶來的合規 narrative
  - MVP 已可運作

### Cyberport CSI / Incu-Tech

- 重視：**技術深度、規模化潛力、團隊背景**
- **OpenTrade 敘事重點**：
  - DDD + Modular Monolith 架構成熟度
  - Smart contract 有第三方 audit
  - AWS 級基礎設施
  - 退休 SFC 高層 + CFD 業界團隊
  - 已有真實鏈上資料

### HKSTP Incu-App

- 重視：**有實際付費客戶 / Ecosystem 整合**
- **OpenTrade 敘事重點**：
  - 已有商戶 / KOL 認領
  - B 端 SaaS 工具費已開始收
  - 與 SFC 監管 / 消委會合作可能性

## 注意事項

### 不可寫入申請材料的內容

- ❌ 提到「token / ICO / 發幣」（觸 VATP 紅線）
- ❌ 任何投資建議性質的功能描述（觸第 4 類牌照紅線）
- ❌ 與 WikiFX 等競品的人身攻擊言論（只能講機制差異）
- ❌ 未經授權的監管機構 logo / 背書
- ❌ 對未持牌商戶的負面評論截圖

### 必須包含的免責聲明

- 平台為純技術服務商，不提供投資建議
- 鏈上資料由用戶自發提交，平台無權刪除
- 所有評論代表用戶個人立場
